# Code Of Conduct

## Our Pledge

We pledge to make participation in this project respectful and harassment-free for everyone.

## Expected Behavior

- Use welcoming language.
- Respect differing viewpoints and experience levels.
- Give and accept constructive feedback.
- Focus on what is best for the community and project.

## Unacceptable Behavior

Harassment, discriminatory language, personal attacks, and publishing private information without permission are not acceptable.

## Enforcement

Maintainers may remove comments, issues, pull requests, or access when behavior violates this code.
