# Security Policy

## Supported Versions

| Version | Supported |
| ------- | --------- |
| 0.1.x   | Yes       |

## Reporting A Vulnerability

Please open a private security advisory on GitHub for sensitive reports. Include reproduction steps, affected commands, and whether a report export can leak data.

## Local Data Boundary

Repo Stratigrapher is local-first. The CLI reads the repository path you provide and writes reports to the output directory you provide. GitHub Pages demos use synthetic fixture data.
